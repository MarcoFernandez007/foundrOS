# foundrOS — Autonomous Startup Engine

A zero-human, AI-powered startup operating system that autonomously builds, markets, and incorporates businesses.

## Features

- **Business Builder** — Describe an idea and watch 17 specialized AI agents build it live
- **Gstack Pipeline** — Every task runs through Plan → Execute → Review → Release
- **Paperclip Orchestration Layer** — Discovery → Product & Build → Go-To-Market → Validation & Release
- **Iterative Autoresearch** — Each iteration improves system prompts and agent markdown prompt files
- **Google Co-Scientist Engine** — AI-powered hypothesis generation with Generator vs Critic debates and Elo tournament ranking
- **Hexa SAI Self-Improvement** — The system evaluates its own output and mutates hyperparameters to improve
- **Corporate Suite** — Autonomous finance projections, cap table, incorporation wizard, and legal document generation
- **3D Visualizations** — Three.js RTS orchestration map and Agent Office scene with animated FoundrOS agents
- **Real AI Integration** — Powered by Gemini 2.5 Flash (with fallback) for code generation, document drafting, and innovation debates
- **GitHub Publishing** — Push generated apps directly to GitHub via the REST API

## Getting Started

1. Open `index.html` in your browser
2. Create an account or sign in
3. Go to **⚙ API Settings** and enter your Gemini API key
4. (Optional) Add a GitHub PAT to enable direct code publishing

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Three.js for 3D visualizations
- Chart.js for financial charts
- Gemini REST API for AI features
- GitHub REST API for publishing
